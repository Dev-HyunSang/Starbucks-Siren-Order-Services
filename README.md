## Functions
### `/register`
#### Request
```json
{
    "name": "HyunSang Park",
    "nickname": "박현상",
    "birthday": "2006-01-02T00:00:00Z",
    "email": "parkhyunsang@kakao.com",
    "password": "helloworld!"
}
```

#### Response
```json
{
    "exp": "2021-12-12T17:02:00.486146+09:00",
    "isOk": true,
    "status": 200
}
```
